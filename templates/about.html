About Geometry Calculator

Geometry Calculator is a lightweight educational web app designed to help students, teachers, and learners calculate the properties of common 2D and 3D shapes quickly and clearly.

The app includes interactive calculator pages for rectangles, squares, triangles, circles, ellipses, rhombuses, trapeziums, pentagons, hexagons, cubes, cuboids, cylinders, cones, and spheres. Each page provides formulas, input fields, and visual previews to make learning more engaging.

Built with Python, Flask, Bootstrap, and SVG, this project focuses on simplicity, responsiveness, and a polished user experience.
